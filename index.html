周数计算示例
// 设置起点日期为开学日
起点日期 = 2026-02-23 (开学)
当前日期 = 2026-03-15 → 显示: Week 3
当前日期 = 2026-02-20 → 显示: before week1
计算公式: 周数 = ⌊(当前日期 − 起点日期) / 7 天⌋ + 1

常见问题

快速解决使用疑问

macOS 安全警告？

由于应用未签名，macOS 会显示安全警告。前往 系统设置 → 隐私与安全性，找到 WeekBar 并点击"允许"。

Windows 风险提示？

Windows Defender 可能标记为"不常见文件"。点击"更多信息" → "仍要运行"，或从安装目录直接运行。

周数显示不正确？

检查起点日期设置，确保当前日期在起点之后。如需修改，重新设置起点日期即可。

如何更新版本？

访问 GitHub Releases 页面下载新版安装包，覆盖安装即可保留原有设置。
