01 / Features 核心特性

架构特点

WeekBar 拒绝 Electron 等重型框架，坚持原生路线。macOS 端基于 SwiftUI 与 AppKit 混合架构， 直接使用系统菜单栏 API；Windows 端基于 WinForms 与原生 Win32 API，实现贴近任务栏的轻量化悬浮层。 两端均采用最小化后台运行策略，仅在必要时刷新 UI，确保资源占用可以忽略不计。

05 / FAQ 常见问题

你可能想知道的问题

是的，WeekBar 完全免费且开源。你可以在 GitHub 上获取源代码，自由使用、修改和分发。

当前版本的安装包尚未进行代码签名，因此 macOS 和 Windows 可能会在首次运行时弹出安全提示。这是正常现象，你可以在系统设置中允许运行。我们计划在后续版本中加入签名支持。

Windows 版本的悬浮条不是系统时钟原生控件，而是贴近任务栏时间区域的轻量显示层。这样可以确保在不同 DPI 设置和任务栏位置下都能正确显示。

点击菜单栏/任务栏中的 WeekBar 图标，选择"设置"或"偏好设置"，在起点日期选项中选择你想要的日期即可。设置会被自动保存，重启后依然有效。

macOS 版本支持 macOS 11 (Big Sur) 及以上版本；Windows 版本支持 Windows 10 和 Windows 11。请确保系统已更新到最新补丁以获得最佳体验。
